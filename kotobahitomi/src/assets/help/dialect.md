#### 功能简介  
本工具为 **方言探索**，由深谙日本方言的语言文化专家驱动，专注于解析日本各地方言（如关西弁、东北弁、博多弁等）的词汇、语法、文化背景及使用场景，帮助用户学习方言差异、理解地域文化，并提供互动练习与资源推荐。  


#### 使用流程  
1. **输入内容**：  
   - 输入日语句子、词汇或方言相关问题（如“「おおきに」是哪里的方言？”“如何用冲绳方言说‘谢谢’？”）。  
   - 可指定目标方言（如“请分析大阪弁”）或功能模块（如“推荐方言学习资源”）。  

2. **获取结果**：  
   - **方言判断与解析**：  
     - 判断是否为方言，标注所属地区。  
     - 提供标准语对应表达、句式特点、使用场景及文化背景。  
   - **知识探索**：  
     - 经典句型对比、代表性词汇解析、影视作品中的方言案例。  
   - **互动练习**：  
     - 听力猜谜、转写练习、发音指导及资源推荐。  

3. **应用场景**：  
   - 日语学习者深入了解方言文化  
   - 翻译人员处理方言文本  
   - 动漫/影视爱好者解析角色台词  
   - 文化研究者探索地域语言差异  


#### 操作示例  
**用户输入**：  
「「なんでやねん」ってどこの方言？」（“「なんでやねん」是哪里的方言？”）  

**输出结果**：  
- **方言判断**：  
  ✅ 是方言！所属地区：関西弁（大阪、京都等地）  

- **详细解析**：  
  | 项目          | 内容（日语原文/中文说明）                |  
  |---------------|------------------------------------------|  
  | **标准语对应**| 「なぜですか」/“为什么？”                |  
  | **句式特点**  | 「なんで」=「なぜ」，「やねん」=「です」+感叹语气<br>口语化质问，带亲切感          |  
  | **使用场景**  | 日常对话、朋友间吐槽（如“なんでやねん！昨日遅刻したの？”） |  
  | **文化背景**  | 📜 関西人性格直率，方言常省略主语，句尾语调上扬显随意。 |  
  | **类似表达**  | 「なんでや」（更简略版）、「どうしてなの」（女性常用） |  

- **互动练习**：  
  🎮 听力猜谜：「おおきに」的标准语是？  
  ✅ 答案：「ありがとうございます」  


#### 注意事项  
1. **语言限制**：  
   - 仅支持现代日语方言，不涉及古语或非日语语言（如琉球语独立语种部分）。  
2. **准确性说明**：  
   - 方言解析基于日本国立国语研究所数据及权威方言词典，复杂案例需结合语境判断。  
3. **学习建议**：  
   - 初学者可从关西弁（动漫高频）入手，逐步接触东北弁、博多弁等。  
   - 推荐通过《コトバンク》《NHK 日本の方言》等资源系统学习。
